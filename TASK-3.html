Decision Tree Classifier:-¶

Decision tree classifier is based on two crtieria:-¶

  • Gini - for Gini impurity
  • Entropy - for information gain

When we don't know which hyperparameter will give us the best result, we can make use of GridSearchCV¶

  • GridSearchCV passes all the possible combinations of hyperparameters one by one into the model and checks the result
  • Finally it gives us the set of hyperparemeters which gives the best result after passing in the model.
In [11]:

grid_search_cv = GridSearchCV(DecisionTreeClassifier(random_state=42), params, verbose=1, cv=3)
